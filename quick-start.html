xiVoodoo Quick Start

Get up and running with dgVoodoo2 in minutes!

1

Download & Install

Download xiVoodoo from the releases page and run the installer as administrator.

Important: Always run xiVoodoo as administrator for proper installation.
2

Set FFXI Path

Open xiVoodoo and go to the Installation tab. Set your Final Fantasy XI installation directory:

C:\Program Files (x86)\PlayOnline\SquareEnix\PlayOnlineViewer

Use the Browse button to find the correct folder if needed.

3

Install dgVoodoo2

Click the green "Install dgVoodoo2" button. The system will automatically install the necessary components and verify the installation.

Status Monitor: Watch the status indicators to ensure everything is properly installed.
4

Configure Settings

Switch to the Configuration tab and adjust settings for optimal performance:

  • Resolution: Set to your monitor's native resolution
  • Anti-aliasing: Start with 2x or 4x MSAA
  • Anisotropic filtering: Set to 8x or 16x
  • dgVoodoo Watermark: Enable to verify it's working
5

Save & Test

Click "Save Configuration" and launch Final Fantasy XI. You should see the dgVoodoo watermark in the bottom-right corner if enabled.

Success! If you see the watermark, dgVoodoo2 is working correctly.
Troubleshooting: If the game crashes or doesn't start, try:
  • Disabling the watermark
  • Reducing anti-aliasing settings
  • Running FFXI as administrator
  • Using the "Verify Installation" button
