What is xiVoodoo?

xiVoodoo is a modern, user-friendly configuration manager for dgVoodoo2, specifically designed for Final Fantasy XI players. It provides an intuitive interface for managing DirectX emulation settings without the complexity of manual configuration files.

Key Features

  • One-click dgVoodoo2 installation
  • Visual configuration interface
  • Real-time system status monitoring
  • Automatic configuration validation
  • Backup and restore functionality
  • GPU and monitor detection
  • Modern, responsive UI
Note: xiVoodoo requires administrator privileges to install dgVoodoo2 components and modify system files.

System Requirements

  • Windows 10/11 (64-bit)
  • DirectX 11 compatible graphics card
  • Final Fantasy XI installed
  • Administrator privileges
  • 4GB RAM minimum
  • 100MB free disk space
Important: Always run xiVoodoo as administrator to ensure proper installation and configuration.

4 Configure Settings

Switch to the Configuration tab to customize dgVoodoo2 settings for optimal performance with Final Fantasy XI.

Configuration Guide

General Settings

  • dgVoodoo Watermark: Enable to show the dgVoodoo logo in-game
  • Force vSync: Prevents screen tearing
  • Keep filter if point sampled: Maintains texture filtering

DirectX Settings

  • Resolution: Set your desired game resolution
  • Fullscreen: Choose between windowed, fullscreen, or borderless
  • Anti-aliasing: Configure MSAA levels
  • Anisotropic filtering: Improve texture quality

Advanced Settings

  • Bilinear DD/CPU operations: Use bilinear filtering for 2D operations
  • Apply Phong shading: Enable smoother lighting calculations
  • Fast video memory access: Optimize memory access patterns

Troubleshooting

Common Issues

Application won't start:

Ensure you're running xiVoodoo as administrator. Right-click the shortcut and select "Run as administrator".

Installation fails:

Check that your antivirus isn't blocking the installation. Add xiVoodoo to your antivirus exclusions if necessary.

Game crashes:

Try disabling the dgVoodoo watermark and reducing anti-aliasing settings. Some configurations may not be compatible with your system.

Verification

Use the "Verify Installation" button to check if all components are properly installed and configured. The system status monitor will show real-time information about:

  • Directory validity
  • dgVoodoo2 DLL presence
  • Configuration file status

Advanced Usage

Manual Configuration

For advanced users, you can manually edit the dgVoodoo configuration file. It's located at:

[FFXI Directory]\dgVoodoo.conf

Backup and Restore

xiVoodoo automatically creates backups of your configuration. You can restore previous settings using the "Reload from File" button in the Configuration tab.

Performance Optimization

  • Enable "Fast video memory access" for better performance
  • Use 2x or 4x MSAA for optimal quality/performance balance
  • Disable the watermark for maximum performance
  • Set anisotropic filtering to 8x or 16x for best texture quality
Tip: The configuration is automatically cached and loaded on startup. Use "Clear Config Cache" to reset to defaults if needed.
